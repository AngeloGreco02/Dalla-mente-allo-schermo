Dalla mente allo schermo

Sito interattivo tratto dalla tesi di Angelo Greco su neurodivergenze, disabilità invisibili, ADHD, DOC e media digitali.

## File da caricare su GitHub

Carica nella root della repository questi file e cartelle:

- `index.html`
- `styles.css`
- `app.js`
- `assets/`
- `vercel.json`

Il PDF della tesi è incluso in `assets/tesi-angelo-greco.pdf`.

## Deploy su Vercel

1. Crea una nuova repository GitHub.
2. Carica tutti i file di questa cartella nella root della repository.
3. Vai su Vercel e scegli `Add New Project`.
4. Importa la repository GitHub.
5. Lascia `Framework Preset` su `Other`.
6. Lascia vuoti `Build Command` e `Output Directory`.
7. Clicca `Deploy`.

Se invece metti questi file in una sottocartella della repository, in Vercel imposta quella sottocartella come `Root Directory`.
